# Penguin

**Generators and utilities for developers** — a collection of handy tools for developers, all in a single HTML page.

This is the source repository for [penguin.web.ap.it](https://penguin.web.ap.it).

![Penguin](https://img.shields.io/badge/version-1.0-neon?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)

## Features

Penguin offers 18+ utilities and generators ready to use, with a dark terminal-style interface (neon green on black theme).

### Available generators and utilities

| Tool | Description |
|------|-------------|
| **Password** | Secure password generator with configurable length and character sets |
| **Username** | Generate valid Linux usernames from domain names |
| **Bcrypt** | Hash and verify passwords with bcrypt |
| **APP_KEY** | Generate application keys (e.g. Laravel) |
| **Hash** | Generate hashes (MD5, SHA-1, SHA-256, etc.) |
| **JWT** | Decode and display JWT tokens |
| **UUID** | Generate UUID v4 |
| **JSON-LD** | JSON-LD utilities |
| **Fake ID** | Generate fake data (name, email, address, etc.) |
| **Base64** | Base64 encode and decode |
| **URL Parse** | URL parsing and analysis |
| **Slug** | Generate URL-friendly slugs from text |
| **Crontab** | Cron expression builder with human-readable explanation |
| **Timestamp** | Unix timestamp ↔ readable date converter |
| **Lorem** | Lorem ipsum text generator |
| **cURL** | Generate cURL commands from parameters |
| **SQL** | SQL utilities |
| **Linux** | Linux commands and utilities |
| **Chmod** | Chmod permissions calculator (numeric and symbolic) |

## Usage

1. Open `index.html` in your browser
2. Use the sidebar to navigate between tools
3. Click on generated values to copy them to the clipboard

No installation, no server, no dependencies — works offline.

## Requirements

- A modern browser (Chrome, Firefox, Safari, Edge)
- No external dependencies
